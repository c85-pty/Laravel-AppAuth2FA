# Laravel-AppAuth2FA
Proyecto en Laravel para la autenticación 2FA con el correo y el lector de qr autenticator (cualquier app a fin a esta implementación).

Este proyecto consiste en un aplicativo web autenticador basado en Laravel para los usuarios (en los casos que activen el doble factor de autenticación con código QR).

Luego de iniciar y loguearse, se le solicitará el código que se genera en qr del método 2FA, el cual con apoyo de un app en android (ejemplo.
